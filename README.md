
# Sodeva - Plateforme de gestion et de formation

![Certifié Qualiopi](./public/images/logos/qualiopi.png)

Plateforme de gestion et de formation pour organismes de formation, certifiée Qualiopi. Sodeva propose des pages marketing, une documentation complète, et une interface de gestion moderne adaptée aux besoins des centres de formation.

## Pourquoi Sodeva ?

Sodeva va au-delà des simples pages de présentation : vous bénéficiez d'un tableau de bord complet, d'une gestion des formations, des apprenants, et d'une expérience utilisateur optimisée pour les organismes certifiés Qualiopi.

## Fonctionnalités principales

- Gestion des formations, sessions, et apprenants
- Pages marketing personnalisables
- Documentation intégrée
- Interface dashboard moderne (Astro, TypeScript, Tailwind CSS)
- Support du mode sombre
- Composants réutilisables (Statistiques, Tableaux, Modals, etc.)
- SEO optimisé et accessibilité renforcée
- Certifié Qualiopi : conformité aux exigences réglementaires

## Démarrage rapide

1. Clonez le projet :

```bash
git clone <votre-repo-sodeva>
cd sodeva
pnpm install
```

2. Lancez le serveur de développement :

```bash
pnpm dev
```

Votre site sera accessible sur [http://localhost:4321](http://localhost:4321)

## Personnalisation

1. **Configuration du site** : modifiez les métadonnées dans `src/config/site.ts`
2. **Design** : adaptez les couleurs et styles dans `src/styles/global.css`
3. **Contenus** : ajoutez vos formations, pages, et témoignages
4. **Dashboard** : connectez votre API ou solution d'authentification
